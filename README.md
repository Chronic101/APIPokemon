# APIPokemon

API REST desarrollada con **Python**, **FastAPI** y **MongoDB**, que consume datos de la API de Pokemon y los almacena en una base de datos local.
Se pueden agregar, eliminar o buscar los movimientos de un pokemon determinado si no se desea saber sus otras características.

---

## Tecnologías utilizadas

- Python
- FastAPI
- MongoDB
- Swagger

---

## Instalación


```bash
git clone https://github.com/Chronic101/APIPokemon.git
cd APIPokemon
pip install -r requirements.txt
```

---


## Ejecución
```bash
uvicorn app.main:app --reload
```
